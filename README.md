🎬 MovieMayhem

🔗 **Live Demo:** [moviemayhem-six.vercel.app](https://moviemayhem-six.vercel.app)

A modern movie discovery and browsing platform designed to help users explore trending films, popular titles, and detailed movie information in a clean, responsive interface.

✨ Overview

MovieMayhem is a frontend-focused movie discovery application that allows users to browse trending films, explore categories, and view detailed movie information in a sleek, modern UI.

The goal of the project is to replicate a production-level entertainment platform experience while demonstrating strong frontend development skills, API integration, and responsive UI design.

🚀 Features
🎥 Browse trending and popular movies
🔎 Search functionality for finding specific titles
📊 Dynamic movie data fetched from external API
🧭 Clean navigation and category-based browsing
📱 Fully responsive design (mobile + desktop)
⚡ Fast loading with optimized rendering
🎨 Modern UI with focus on user experience and visual clarity
🧩 Component-based architecture for scalability


🛠️ Tech Stack
Framework: React
Styling: CSS3
API: Movie database API
Deployment: Vercel
Language: JavaScript


## Getting Started
```bash
npm install
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to view the app.
